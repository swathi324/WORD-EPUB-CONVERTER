# 📖✨ Word-to-EPUB Converter 🚀

> Transform Microsoft Word documents into beautifully formatted, production-ready EPUB ebooks with smart cleaning, advanced typography, and modern customization tools.

---

# 🌟 Overview

The **Word-to-EPUB Converter** is a modern full-stack web application built for authors, publishers, students, and digital content creators who want to generate clean and professional EPUB files from Word documents effortlessly.

This tool not only converts `.docx` files into EPUB format but also provides:

✅ Smart document cleaning
✅ Typography customization
✅ Metadata management
✅ Chapter organization
✅ Cover image integration
✅ Live preview experience

---

# 🎯 Key Features

## 📄 DOCX → EPUB Conversion

Easily upload Microsoft Word files and convert them into high-quality EPUB ebooks.

✨ Supports:

* `.docx` files
* Structured content conversion
* Clean HTML generation

---

# 🧹 Smart Content Cleaning

Powered by `mammoth.js` for extracting clean HTML from Word documents.

### Automatically Removes:

❌ Junk inline styles
❌ Empty tags
❌ Unnecessary formatting
❌ Word-specific clutter

### Result:

✅ Clean EPUB structure
✅ Better readability
✅ Optimized ebook formatting
